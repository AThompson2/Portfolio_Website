Nomad Camper Damper Website

This is my first attempt at a complete website. It was a steep learning curve, but a great way to learn and gain experience in web development.

The Project is pseudo website that i built for a friends start-up business idea. The business is called "The Nomad Camper Damper". The Nomad Camper Damper is a future range of baking products that are designed to be quick and easy to cook in a campfire whilst camping.

The websites purpose it to provide information about the company, the products, contact information and an eccomerce store for purchasing the products online.

So with all that in mind i have created the website as you will see. There are quite a few bugs which i will discuss later. More Javascript and an increase in security needs to be added. This is version 1.0 of the website with a version 2 comming soon.
